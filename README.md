# EventPulse

Create polished, shareable calendar invites in seconds.

EventPulse is a mobile-friendly web app that helps you:

- Build event invites quickly.
- Download `.ics` files.
- Open Google Calendar with one click.
- Generate and download QR codes for invites.
- Share events directly from supported devices.

## App Showcase

### What makes EventPulse useful

- **Fast planning:** Choose a start time and the app auto-sets end time to +1 hour.
- **Cleaner sharing:** QR and Google links include title, date/time, location, description, and timezone.
- **Modern UI:** Responsive, card-based layout with a live preview panel.
- **Practical actions:** Symmetric action buttons for download, Google, QR, share, and reset.

### Core user flow

1. Enter event details (title, time, location, description).
2. Download an `.ics` invite or open Google Calendar.
3. Generate a QR code and share/download it.

## Quick Start

Run locally with any static server:

```bash
python3 -m http.server 4173
```

Open:

```text
http://localhost:4173
```
